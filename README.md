🔧 Tecnologias Utilizadas:

    🌐 HTML5
    
    🎨 CSS3

    💻JavaScript
    
🌟 Destaques:

    🎟️ Interface intuitiva para gerenciar sorteios
    
    📱 Design responsivo e acessível
    
    🗳️ Funcionalidades eficazes para sorteios e eventos
